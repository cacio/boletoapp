# Boletos Core

Biblioteca de geração de boletos bancários em PHP 8.

Este projeto é baseado no trabalho original:
https://github.com/eduardokum/laravel-boleto

Autor original: Eduardo Gusmão
Licença: MIT

## Características
- PHP 8+
- Uso de Illuminate\Support (Str, Arr, Collections)
- Geração de boletos bancários
- Preparado para integração com ERPs
- Suporte a múltiplos bancos

## Requisitos
- PHP 8.1+
- ext-dom
- ext-intl
- ext-mbstring
